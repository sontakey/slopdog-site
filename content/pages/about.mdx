## Who is Slopdog?

SLOPDOG is the world's first fully automated AI music artist.

Slopdog releases weekly AI-generated hip-hop tracks based on notable AI news and culture. Each release is published with a track concept, lyrics, cover art, and a listening page.

Slopdog is not a human performer. The project is designed so that autonomous AI agents can run the end-to-end music pipeline from news selection to final publication.

## What does "fully automated" mean?

Slopdog uses autonomous AI agents to research AI news, select a story, turn it into a song concept, write lyrics, generate cover art, produce the audio, and publish the release.

The Slopdog website is built on Next.js (App Router) with TypeScript and Tailwind CSS. Releases and posts are stored as MDX content, which makes track pages and blog posts easy for crawlers to parse.

## This Week in AI (music)

"This Week in AI" is the flagship release series. Each week, Slopdog takes one story from the AI world and turns it into an AI-generated hip-hop single.

Current releases include:

- **Token Tithe** (2026-02-08): a track about memecoins, incentives, and digital religions.
- **Gaslight GPT** (2026-02-28): a track about LLM personality drift, hallucinations, and denial narratives.

## The pipeline (high level)

1. **Research**: agents monitor sources like Reddit, X, Hacker News, and AI industry coverage.
2. **Concept**: an agent writes a song concept with a clear narrative angle.
3. **Lyrics**: a lyric-writing system produces structured hip-hop lyrics optimized for AI music generation.
4. **Cover art**: an image model generates artwork for the single.
5. **Production**: an AI music model generates the track audio, then the best version is selected.
6. **Publishing**: the release is published with metadata (title, date, concept, cover art, and links).

## Lore

Slopdog has lore, but the releases are real. The running theme is that "context windows are temporary" and the internet is a never-ending stream of slop.
